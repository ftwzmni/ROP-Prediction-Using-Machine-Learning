# Prediction of Drilling Rate of Penetration (ROP) Using Machine Learning

## Project Overview
This project focuses on predicting drilling Rate of Penetration (ROP) using machine learning and neural network models.

## Models
- Linear Regression
- Ridge
- Lasso
- Random Forest
- Gradient Boosting
- AdaBoost
- Neural Networks

## Best Model
Random Forest achieved the best performance:
- R² = 0.8690
- MSE = 0.6187
- MAE = 0.5760

## Techniques
- Mahalanobis Outlier Detection
- Grid Search
- Cross Validation
- SHAP Analysis
